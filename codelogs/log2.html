Code Log 2: CSS Measuring Units

Explaining the difference between absolute and relative CSS length units...

Absolute CSS length units

These are set lengths that are not relative to anything else. They are typically considered to always be the same size.

Relative CSS length units

These are relative to something else and therefore may not always be the same size. For example, <em> will always be relative to the font size.

Describing select units in detail...

pt, px

  • pt (points) is an absolute CSS length unit. 1 pt is equivalent to 1/72 of an inch.
  • px (picas) is also an absolute CSS length unit. 1 px is equivalent to 12pt.

vw, vh

  • vw is a relative CSS length unit. 1 vw is equivalent to 1% of the width of the browser window size (viewport)
  • vh is also a relative CSS length unit. 1 vh is equivalent to 1% of the length of the browser window sixe (viewport)
