Code Log 6: Code References

What did you look up?

I looked up information on the <div> tag.

What sites did you consult?

I used W3Schools, Mozilla, HTML.com, and FreeCodeCamp.

What are the differences in the information presented on each site?

W3Schools

W3Schools was overall pretty thorough on what a div tag is, how to use it, when to use it, and many other details about styling and formatting a <div> tag. There's lots of great examples with the option to "try it yourself".

Mozilla

Mozilla also gave a good basic description of what a <div> tag is and where to use. They also touched on some styling aspects and, more noteably, the accessibility of the <div> tag.

HTML.com

HTML.com provided a very brief, very generic description of a <div> tag. They provided no information on styling at all.

FreeCodeCamp

FreeCodeCamp provided a good description of a <div> and how to style it. They even delved more into how a <div> can be used to make artwork through styling.

What features do you think a code reference page should offer?

I think a good reference page should include the following features:

  • A thorough description of the code being referenced. What is it? How is it used? Where is it appropriate and inappropriate to use it?
  • An example of the code being used. Both what it looks like as code and what the end result looks like. Bonus points for including a "try it yourself" feature.
  • Styling options (if applicable). Doesn't need to be every single option known to man, but maybe the top ones that are most commonly used or are most useful.
  • Bonus points for including the accessibility of the code.

Do any of your 4 resources stand out as particularly good or bad?

Overall, I think Mozilla did the best job of providing the most information both on the page and with links to relevant code. HTML.com was the worst. For a beginner, I think it leaves more questions than answers with no additional navigation options to learn more besides their search bar. Which if I knew nothing about code, I probably wouldn't even know what to search for.
