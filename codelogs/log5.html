Besides background-image, what other CSS properties affect the display of background images?

background-size

This property changes the size of the background image. This can be specified in lengths, percentages, or by using one of the keywords (auto, contain, or cover).

background-position

This property adjusts the starting point of a background image. The default is the top-left corner of an element, but can be changed with keywords (left top, left center, left bottom, right top, right center, right bottom, center top, center center, center bottom), percentages (x% y%), using values for x and y in units such as px.

background-repeat

This property sets how/if a background image will repeat. By default, background images repeat vertically and horizontally.

background-attachment

This property controls if a background image is fixed or scrolls with the rest of the page.

In one sentence, how would you define the differences between dimensions and resolution in a bitmap image?

Dimensions determine the total number of pixels in a bitmap image (overall size) while resolution determines how those pixels are distributed across a bitmap image (density).
